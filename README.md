
**NotebookLM Clip** helps you export content, well-formatted Markdown plus optional chats, sources, JSON, and assets, from **NotebookLM** into files you can keep, view locally, and update as needed.

## Feedback welcome

We’re continuously improving the user experience—your suggestions are appreciated!


### What it exports

- **Notebook Markdown**: the main formatted notebook content
- **Chats Markdown** (optional): your chat history for the notebook
- **Sources Markdown** (optional): extracted sources content
- **JSON bundle** (optional): a structured export of fetched data
- **Assets** (optional): slide decks, audio, video, quiz/flashcards HTML (when available)
- **Images** (optional): images from slide decks and infographics

### How to use

1. Open a notebook at `https://notebooklm.google.com/notebook/<notebook-id>`
2. Click the extension icon to open the popup
3. Choose download options (folder + toggles)
4. Click **Fetch & Download**

Files are saved into your browser’s Downloads location under the folder you choose.

### Requirements

- You must be signed in to NotebookLM in the same browser profile.
